# 🦑 ClawDocs AI

**"你的第二个脑子"** — 用一句话需求，生成可直接使用的合同、报告、方案、邮件。

---

## 📁 公司文件结构

```
clawdocs-ai/
├── README.md              ← 你在这里
├── 商业计划书.md          ← 完整BP（市场、财务、上线计划）
├── 公司章程.md            ← 使命、价值观、产品原则
├── 推广计划.md            ← 增长策略、渠道、KPI
├── 竞品分析.md            ← 深度竞品对比、SWOT
├── 路演PPT.html           ← 投资人演示（浏览器打开）
├── index.html             ← 公司官网（浏览器打开）
├── logo.svg               ← 公司Logo
└── 内容素材/
    └── 小红书-3分钟写合同.md   ← 推广文案素材
```

## 🚀 产品核心

**一句话 → 可签字的文档**

- 目标用户：中小企业主、自由职业者、电商卖家
- 主打功能：合同起草、报告撰写、邮件助手、方案输出
- 定价：免费版 + ¥49/月个人版 + ¥199/月团队版
