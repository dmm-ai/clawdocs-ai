# ClawDocs AI - AI文档工厂

🦑 **ClawDocs AI** — 量子文档智能有限公司旗下产品

一句话需求，自动生成合同、报告、方案、邮件。

## 功能
- 📄 AI合同起草（咨询服务/采购/劳动等）
- ✉️ AI邮件助手（跟进/道歉/报价等）
- 📊 AI报告撰写（月度/季度工作汇报）
- 💡 AI方案输出（商业提案/培训方案）
- ✨ 文档润色优化

## 立即体验
👉 https://clawdocs-ai.vercel.app/mvp.html

## 技术栈
- 前端：HTML5 + CSS3 + Vanilla JS（无框架依赖）
- AI后端：OpenRouter API（支持 Gemini/Qwen 等）
- 部署：Vercel
